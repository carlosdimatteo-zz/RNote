# RNote 

An app to create and manage Notes

## Prefect for:
  * ToDos 
  * Activities
  * Shopping Lists
  * Birthday reminders
  * School Homework tracking
  * And much, much more...

Created with [React Native](https://facebook.github.io/react-native)
